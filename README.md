## Start project
- Download this branch repo https://github.com/rodolphe37/opia-react-template.git
- Do in the console :
-    ```$ cd opia-react-template```
-    ```$ npm i``` or ```$ npm install ```
to install the dependencies and
-    ```$ npm start```

Enjoy !
